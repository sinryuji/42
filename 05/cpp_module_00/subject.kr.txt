--------------------------------------------------------------------------------

Warlock 클래스를 만들어라. 이것은 코플린 폼이어야 한다.

이 것은 다음과 같은 프라이빗 속성이 있다:
* 이름(string)
* 제목(string)

프라이빗에 대해 게터 함수를 작성해야 한다:
* getName, returns a reference to constant string
* getTitle, returns a reference to constant string

두 함수는 모두 constant Warlock에서 호출 할 수 있어야 한다.

다음 setter를 만들어라:
* setTitle, returns void and takes a reference to constant string

코플린 양식 외에도 이름과 제목을 순서대로 받는 생성자를 갖게 될 것이다.
이름과 제목 없이 Warlock을 복사하거나 복사 하여 인스턴스화 하거나, 인스턴스화 할 수 없다.

For example :

Warlock bob;                            //Does not compile
Warlock bob("Bob", "the magnificent");  //Compiles
Warlock jim("Jim", "the nauseating");   //Compiles
bob = jim;                              //Does not compile
Warlock jack(jim);                      //Does not compile

생성시 Warlock은 다음과 같이 출력한다.

<NAME>: This looks like another boring day.

물론, <NAME>은 객체의 name으로 치환 되어야 한다.

소멸시에는 다음과 같이 출력한다.

<NAME>: My job here is done!

우리의 Warlock은 스스로를 소개 할 수 있다.

다음 함수를 작성하라:
* void introduce() const;

다음과 같이 표시되어야 한다.

<NAME>: I am <NAME>, <TITLE>!

메인문 예시가 있다 출력이 비슷해야 한다.

int main()
{
  Warlock const richard("Richard", "Mistress of Magma");
  richard.introduce();
  std::cout << richard.getName() << " - " << richard.getTitle() << std::endl;

  Warlock* jack = new Warlock("Jack", "the Long");
  jack->introduce();
  jack->setTitle("the Mighty");
  jack->introduce();

  delete jack;

  return (0);
}
